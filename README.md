# SPOTS Pitchdeck Website

NDA-gated pitch deck website for SPOTS app using Next.js, Supabase, Inkless (FREE e-signatures), and Resend.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Copy environment variables:
```bash
cp .env.local.example .env.local
```

3. Fill in your environment variables in `.env.local`

4. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

## Project Structure

- `/app` - Next.js App Router pages and API routes
- `/lib` - Utilities, integrations, and design tokens
- `/public` - Static assets

## Features

- ✅ NDA signature via Inkless (FREE - unlimited signatures)
- ✅ Secure access token system
- ✅ Session-based authentication
- ✅ Interactive React-based pitch deck
- ✅ Analytics tracking (Supabase)
